# 📊 フローチャートの例

ここではMermaidを使って簡単なフローチャートを表示します。

```mermaid
graph TD
    A[開始] --> B{意思決定?};
    B -- はい --> C[処理 A];
    B -- いいえ --> D[処理 B];
    C --> E[終了];
    D --> E;
```

```mermaid
graph LR
    A[開始] --> B{意思決定?};
    B -- はい --> C[処理 A];
    B -- いいえ --> D[処理 B];
    C --> E[終了];
    D --> E;
    E --> F[hoge];
    F --> G[hoge];
    G --> H[hoge];
    H --> I[hoge];
    I --> J[hoge];
    J --> K[hoge];
    K --> L[hoge];
    L --> M[hoge];
    M --> N[hoge];
```

# 📈 ガントチャートの例

```mermaid
gantt
    title プロジェクト計画
    dateFormat  YYYY-MM-DD
    section 設計
    要件定義     :a1, 2023-01-01, 7d
    システム設計 :after a1, 5d
    section 開発
    フロントエンド開発 :2023-01-08, 8d
    バックエンド開発   :2023-01-10, 8d
    section テスト
    単体テスト     :2023-01-18, 5d
    結合テスト     :2023-01-22, 5d
```
