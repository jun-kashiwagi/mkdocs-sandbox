# 受注売上一覧

---

## 画面説明

受注売上明細を検索・一覧表示するための画面です。

### ユースケース

- 営業担当者が受注売上明細を検索・一覧表示する
- 受注売上明細の詳細情報を確認する
- 受注売上明細の一括修正画面に遷移する
- 出荷確定画面に遷移する
- 発注一覧、在庫一覧、預り品許可一覧などの関連機能に遷移する

### 用途

- 受注売上明細の検索・一覧表示
- 受注売上明細の詳細確認
- 受注売上明細の一括修正への遷移
- 出荷確定への遷移
- 関連機能への遷移

| EDI処理状況からの遷移時 | 部署/部署グループ、売上先/売上先グループのラジオボタンを非活性 |
| 出荷確定テーブル1からの遷移時 | 一部検索項目を非活性、検索条件を自動設定 |
