## 用語集

### 受注・売上関連

??? info "受注"
    顧客からの注文を受けることを指します。システムでは、受注データの登録、確認、修正などの機能が提供されています。

??? info "売上"
    商品やサービスを販売して得た収益のことです。受注データから自動的に売上データが生成されます。

??? info "伝票"
    取引の記録を記載した書類のことです。出荷伝票、売上伝票、手書き伝票など、様々な種類があります。

??? info "EDI"
    Electronic Data Interchange（電子データ交換）の略称です。企業間で標準化された形式でデータを交換する仕組みです。

??? info "帳合"
    複数の取引先との間で、取引内容を照合・調整することを指します。帳合データのアップロードや一覧表示が可能です。

---

### 商品・マスタ関連

??? info "商品コード"
    商品を識別するための一意のコードです。仮商品コードは、正式な商品コードが確定するまでの一時的なコードです。

??? info "商品カタログ"
    取り扱い商品の情報をまとめたカタログです。商品の詳細情報や価格などを確認できます。

??? info "販売単価マスタ"
    商品ごとの販売単価を管理するマスタデータです。店舗や取引先ごとに異なる単価を設定できます。

??? info "商品取引変換マスタ"
    商品コードと取引先コードの対応関係を管理するマスタデータです。異なるシステム間での商品コード変換に使用されます。

---

### 処理・操作関連

??? info "エントリー"
    データをシステムに登録・入力することを指します。受注エントリー、伝票エントリーなどがあります。

??? info "アップロード"
    外部ファイルからデータをシステムに取り込む処理です。受注データアップロード、帳合データアップロードなどがあります。

??? info "重複チェック"
    同じデータが重複して登録されていないかを確認する処理です。エラーが検出された場合は、重複チェックエラー一覧で確認できます。

??? info "出荷確定"
    注文された商品を出荷することを確定する処理です。出荷確定後、仕分指示が発行されます。

??? info "仕分指示"
    倉庫での商品の仕分け作業を指示する情報です。出荷確定と同時に生成されます。

---

### その他

??? info "見積書"
    商品やサービスの見積もりを記載した書類です。見積書一覧で過去の見積もりを確認できます。

??? info "FAX注文用紙"
    FAXで注文を受ける際に使用する用紙です。システムからダウンロードして使用します。

??? info "マイナス利益申請"
    利益がマイナスになる取引について、承認を得るための申請です。マイナス利益申請一覧で管理されます。

??? info "得意先別データファイル"
    取引先（得意先）ごとにデータを出力したファイルです。各得意先向けのデータを個別に生成できます。
